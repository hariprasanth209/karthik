# MERN Learning Journey

Hi! I'm Sam from Los Angeles, documenting my full-stack web dev journey with the **MERN stack** (MongoDB + Express + React + Node.js) starting Feb 2026.

## Objective
Master modern full-stack development: build scalable apps from UI to database, understand real-world flows, and eventually deploy a complete project (maybe a social app, blog, or e-commerce prototype).

## Tech Stack Overview
- **Frontend**: React (Vite for fast setup) + modern hooks, state management (Context or Redux if needed)  
- **Backend**: Node.js (LTS v24) + Express for APIs  
- **Database**: MongoDB (Atlas free tier to start) + Mongoose for modeling  
- **Tools**: Git/GitHub, VS Code, Thunder Client for API testing, JWT + bcrypt for auth
